Instalación:

1. Asegúrate de tener Python instalado en tu sistema.

2. Crea un entorno virtual:
   python -m venv venv
   source venv/bin/activate  # En Windows usa: venv\Scripts\actívate


3. Instala las dependencias necesarias (si hay alguna especificada en un archivo requirements.txt).

Uso:
1. Ejecuta el archivo principal de la aplicación:
   python algoritms.py

2. Selecciona el método de moderación que deseas utilizar, en el selector que se encuentra titulado como método
   de resolución.

3. Carga un archivo de texto que contenga la red social con el formato adecuado con el botón cargar archivo
   Con el botón "Cargar Archivo".

4. Posteriormente, puede moderar la red con el botón "Moderar"

4. Visualiza los resultados en la interfaz gráfica.

5. Puedes exportar un archivo txt con la información sobre la moderación en el botón "Exportar TXT"
